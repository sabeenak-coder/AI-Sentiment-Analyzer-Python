# AI Sentiment Analyzer Documentation

## Objective

To analyze text sentiment using Artificial Intelligence techniques.

## Description

The application uses the TextBlob library to analyze user input and determine whether the sentiment is Positive, Negative, or Neutral.

## Features

- Sentiment score calculation
- Positive sentiment detection
- Negative sentiment detection
- Neutral sentiment detection
- Interactive command-line interface

## Technologies Used

- Python
- TextBlob

## Conclusion

This project demonstrates Natural Language Processing (NLP) concepts by performing sentiment analysis on text using Python.
