## Project Description

This project is an AI-based Sentiment Analyzer developed using Python and the TextBlob library. It analyzes user-entered text and classifies the sentiment as Positive, Negative, or Neutral based on the sentiment score.

---

## Features

- Analyze text sentiment
- Positive sentiment detection
- Negative sentiment detection
- Neutral sentiment detection
- Command Line Interface (CLI)
